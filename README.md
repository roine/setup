# Export

```bash
./export.sh
```

This will update the Brewfile.

# Install

```bash
./install.sh
```

This will:

-   Symlink all the dotfiles to their respective locations. (git, zsh)
-   Change the shell to use ZSH
-   Install Homebrew
-   Install binaries using the Brewfile
-   Install oh my zsh
-   Install p10k
